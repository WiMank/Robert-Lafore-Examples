Lafore's 2-3-4 Tree
2-3-4 Tree Operation

Fill creates a new tree with N nodes.

Ins inserts a new item with value N.

Zoom toggles between close-up of selected nodes and view of complete tree.

Blue triangles indicate a node has children.

Click on a node to see its children, if they're not already in view.

(Type N into "Enter number" box.)
