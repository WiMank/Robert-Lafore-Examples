Lafore's Red-Black Tree
Red-Black Tree Operation

Click on node to move arrow to it

Start makes a new tree with one node

Ins inserts a new node with value N

Del deletes the node with value N

Flip swaps colors between black parent (arrow) and two red children

RoL rotates left around node with arrow

RoR rotates right around node with arrow

R/B toggles color of node with arrow

(Type N into "Number" box.)
